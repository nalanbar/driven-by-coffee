# drivenby.coffee

Static photography portfolio. Powered by plain HTML, CSS, and JS — no build step.

## Publish workflow

See [CLAUDE.md](CLAUDE.md) for the Claude Code publish workflow.

## Local preview

```bash
npx serve .
# or
python3 -m http.server
```

Then open http://localhost:3000 (or 8000).

## GitHub Pages setup

Point GitHub Pages at the repo root (or `/docs` if you prefer).
No Jekyll, no build actions needed.
